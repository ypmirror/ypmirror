TOR Browser Bundle links for all platforms : https://www.torproject.org/download/download-easy.html.en

Windows
On Windows the tor browser bundle its a autoextractible portable version of tor bundled with a browser, everything ready to use. When you finished downloading and you have extracted file somewhere, just go on our .onion website with the new browser window.

Mac OS X
On Mac OS X just launch the Onion icon that appeared into your dock on the Download tab, and just go on our .onion website with the new browser window.

Linux
It's easy as pie! For all of these UNIX-like operating systems you just have to get the "Tor Browser Bundle for 64-Bit Linux" if you're running a 64 bit system, otherwise download the "Tor Browser Bundle for Linux" in case you're using a 32 bit distribution or when you're not sure whether or not you have a 64 bit system on your box. Then simply extract the archive with a program of your choice and run the "start-tor-browser" shell script which is located within the extracted folder. That's all! Just go to our .onion website with the new browser window. If it doesn't work out of the box make sure the script is executable by running the command "chmod a+x /home/username/path_to_the_script".
